类型

字面量类型

字面量类型就是其本身

基本类型

  • 基本类型
    • boolean 类型：支持true，false
    • number 类型：支持浮点数，十进制，二进制0b，八进制0o，十六进制0x，Infinity无穷大，NaN
    • string 类型：支持es6模板字符串
    • null，undefined：null，undefined，null和undefined是任何类型的子类，默认情况下只能赋值给他们自身，如果关闭了--strictNullChecks严格检查标记，则可以赋值给任何类型。
    • void 空值类型：表示没有任何类型，与any相反，一般用于不关心函数的返回值时，则为这个函数定义void类型。

任意类型

  • 类型等级
    1. 顶级类型：any，unknown，可以包含所有类型
    2. Object，除了顶级类型外，包含所有类型
    3. 原始值的构造函数，Number，String，Boolean
    4. 原值值的实例对象，number，string，boolean
    5. 字面量类型，1， '1'，true
    6. never

any

any类型相当于解除了ts对当前变量的类型检测，any类型的变量可以复制给任意变量，会造成其他变量类型的混乱，如果一个变量没有指定任何类型，则默认为any类型（即隐式any类型）

unknown

unknown表示未知类型的值，实际上就是一个类型安全的any，unknown类型的变量不能直接赋值给其他变量，只能赋值给它自身或者any类型

unknown类型定义的对象，无法读取任何属性和方法

函数重载

函数重载：

  • 函数名相同，参数不同，返回值不固定
  • 重载函数来定义类型规则，执行函数执行代码逻辑
  • 参数的类型不同，因此操作函数的参数要定义为any；参数的数量不同，因此操作函数中多出的参数设为可选。

  • 通过key可以获取value，通过value也可以获取key
  • 字符串不支持反向映射
    • 数字类型被编程成js后：Types[Types["success"] = 666] = "success";
    • 字符串被编译成js后：Types["success"] = "666";

Type 类型别名

type 与 interface的区别

  • type不能使用extends去继承，只能使用交叉类型，或联合类型
  • type不能重名
  • interface不能使用交叉类型，或联合类型

never类型

never类型表示无法到达的类型，在联合类型中会被忽略
